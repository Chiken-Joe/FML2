#  Cynical Circle API Automation Tests
Этот проект содержит чистый, модульный и легко расширяемый фреймворк для проверки функциональности API, полного экзистенциального юмора и скрытых багов.


##  Особенности API, которые учтены в тестах
- **In‑memory хранилище** — все данные живут в RAM и сбрасываются при перезапуске сервера. Тесты не полагаются на предзагруженные записи, а динамически создают свои.
- **Отсутствие авторизации на большинстве эндпоинтов** — создание, обновление и удаление пользователей, постов, комментариев доступно без токена (кроме `/auth/logout`, `/auth/me`, `/auth/refresh`). Это архитектурная особенность, которую тесты проверяют и документируют.
- **Тоггл реакций** — одна реакция на пост от пользователя; повтор той же реакции удаляет её, другой тип — изменяет.
- **Экзистенциальные статусы пользователей** — `contemplating_the_void`, `pretending_to_work`, `on_the_verge`, `running_on_caffeine`.
- **Специфические сообщения об ошибках** — API возвращает ироничные тексты в поле `detail`, они проверяются в тестах.
- **Статистика** — рассчитывается на лету и включает среднее число комментариев на пост, самую частую реакцию и самого активного прокрастинатора.

##  Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone <url-репозитория>
cd cynical-circle-tests


### 2. Создайте и активируйте виртуальное окружение
python -m venv .venv
# Windows
.venv\Scripts\activate
# Linux/Mac
source .venv/bin/activate

### 3. Установите зависимости
pip install -r requirements.txt


### 4. Запусти тесты
pytest tests/ -v
